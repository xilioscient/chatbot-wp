# Linee guida di sicurezza

- Usare HTTPS per tutte le comunicazioni.
- Limitare CORS e IP consentiti al servizio LLM.
- Non esporre il servizio LLM pubblicamente senza autenticazione.
- Aggiornare regolarmente dipendenze Python e PHP.
- Monitorare log e impostare rate limiting.
- Proteggere JWT secret e chiavi fallback.
